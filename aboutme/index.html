关于我 -- Pikipity

现就读于“澳门大学” EEE 专业，当年为了可以提高英语水平于是来到了这个全英语教学的地方。学硬件的同时发现自己对于软件方面的东西也爱的一塌糊涂，本身没有什么特别的兴趣爱好，闲着也是闲着，于是凭靠着自己折腾的劲头，学会了很多乱七八糟的语言：C、Python、Java、LaTex、Html、Markdown 等等（我甚至还接触过 Lisp，仅仅是以仰望大神的姿态接触过而已），样样会，但是都没有过于精通。但是我认为学一种编程语言或是熟悉一种操作系统，都是学习一种思想或是一种习惯，编程语言的语法千变万化，但是语法仅仅是思想的一种承载方式，掌握语法背后所体现出来的对程序、人事的理解才是更加重要的。由于好奇心，还曾经在自己电脑上安装过各种各样发行版本的 Linux，深深的被 Linux 简洁的美感所折服，但是在经历过各个 Linux 发行版本的折腾之后，最后选择了使用 Mac OS X（感觉驴唇对不上马嘴 囧），因为感觉这是一个可安逸可折腾的系统，支持上也比 Linux 更丰富。现在，本着“生命不止，折腾不息”的精神继续着自己的折腾事业。

很多人说这个网站名字很傻，我也是多次想要更改，但是想来想去发现还是这个名字比较贴近这个网站的内容，内容虽然广泛，但是都是浅尝则止，没有高深的内容，以原创为主，原创内容也都是简单的小程序和小电路，没有大型长期的项目，所以只是个人的瞎折腾而已。

也有人说，既然浏览量如此惨淡为什么还要继续更新，其实这个小站对于我自己的意义远比对于其他人的意义重大的多，本质上，无论是理论探讨还是小项目发表，每一篇原创文都是我的一个阶段的小结。当然也是将自己仅有的一点的所知所学展示出来，希望能够帮助到看到的读者，也算是这么多年来对网络上那些不知名帮助的回馈。
