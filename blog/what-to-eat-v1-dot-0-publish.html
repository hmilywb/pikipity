'今天晚上吃神马？' 基本完成

Posted by pikipity  
2013年05月22日
Life, Python
Comments

“今天晚上吃神马？” Version 1.0 完成。可以到项目主页下载。下载下来之后，源码、打包之后的程序都在其中，主程序在 “dist” 文件夹中，双击即可使用（注意路径中不要有中文）。

Python 编写，Tkinter 做 GUI，预想中的功能都已经实现：

  1. 从 take away Macao 自动抓取餐厅信息，包括：

    1. 餐厅名
    2. 电话
    3. 附注
    4. 营业时间
    5. 餐厅地址
    6. 菜单网址（点击，从浏览器直接打开）
  2. 随机抽取一个餐厅
  3. 可以设置自己所在地区，根据所在地区抓取不同的餐厅

下一步需要改善的地方：

  1. 找到一种能够释放内存的方法：连续多次点击之后，由于开启过多进程导致内存占用过大
  2. 找到一种抓取图片的好方法，直接抓取菜单并显示（不用打开浏览器了），抓取餐厅标志并显示
  3. 还要改善一下难看的界面
